Unreal Session 1 Mar 8, 2026
Just Getting There — Session 1

A roguelike dark comedy platformer. Everyday life gone wrong.

The Concept

  • Player is a tired everyman trying to complete a mundane task (get to the bus stop, buy milk, return a library book)
  • Each run the level layout shifts randomly — gaps, drifting platforms
  • NPCs wander and block or shove you
  • Environmental chaos escalates each run — falling ceiling tiles, indoor rain
  • Tone: melancholy comedy. Mundane framing of absurd situations.
  • On death: a dry journal entry from your character reflecting on what went wrong

    • What We Built

    A working browser-based prototype in React + HTML5 Canvas. One file: App.jsx.

    Features in the game:

  • Randomly generated levels with ground gaps and floating drifting platforms
  • NPCs — friendly (block you, say passive-aggressive things) and hostile (shove you off ledges)
  • Falling ceiling tiles from run 2 onwards
  • Indoor rain from run 2 onwards
  • Time limit per run — escalates chaos if you linger
  • Death screen with randomized reason + journal entry
  • Win screen with deflating victory message
  • Each run gets harder — more NPCs, wider gaps, faster hostiles

    • Money throwing mechanic:

  • Press F to throw money at NPCs
  • Coins arc through the air with gravity
  • Hit an NPC — they freeze for ~2 seconds
  • Limited to 5 coins per run

    • Dev Environment

  • VS Code + Node.js v24.14.0 (installed via Chocolatey)
  • Vite + React via npm create vite@latest . -- --template react
  • Run with npm run dev → localhost:5173

    • Key Concepts Learned

  • useEffect — runs code when something changes
  • useRef — holds game state without re-renders
  • requestAnimationFrame — drives the game loop
  • ctx.fillRect — draws everything on canvas
  • e.key — which key was pressed
